# Learning Computer Vision 👁️

¡Hola! En este repositorio comparto mi evolución y proyectos en el mundo de la Visión por Computadora, desarrollados para prácticas de estudio personal.

## 🛠️ Tecnologías Utilizadas
* Python 3.x
* OpenCV (`opencv-python`)
* NumPy

## 📁 Estructura del Repositorio

* **`01-open-camera`**: Inicialización y pruebas de la cámara web.
* **`02-using-images`**: Lectura, manipulación y guardado de imágenes estáticas.
* **`03-color-detection`**: Segmentación de imágenes en el espacio de color HSV.
* **`04-face-detection`**: Detección de rostros en tiempo real usando Haar Cascades.
* **`05-face-recognition-dataset`**: Script para recolectar y etiquetar imágenes de rostros para el entrenamiento.
* **`06-model-trainer`**: Entrenamiento de un modelo de reconocimiento facial (LBPH Face Recognizer).
* **`07-face-recognition`**: Sistema de reconocimiento facial en tiempo real utilizando el modelo entrenado.
* **`08-eda`**: Análisis Exploratorio de Datos (EDA) aplicado a los datasets de imágenes.

> ⚠️ **Nota sobre los datos:** Los datasets utilizados (`lfw_funneled` ,carpetas de imágenes y datasets) no se suben al repositorio debido a restricciones de tamaño de GitHub, pero están configurados localmente en una carpeta `data/`.
